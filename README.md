# Курсовая работа «Резервное копирование»

Возможна такая ситуация, что мы хотим показать друзьям фотографии из Интернета, но иногда сайты могут быть недоступны. Давайте защитимся от такого.  
Данная программа предназначеня для резервного копирования картинок с сайта [dog.ceo](https://dog.ceo/dog-api/documentation/) в облачное хранилище Яндекс.Диск.  

### Входные данные:
Пользователь вводит:
1. Породу собаки на английском языке
2. Количество картинок для сохранения
2. Токен с [Полигона Яндекс.Диска](https://yandex.ru/dev/disk/poligon/).
